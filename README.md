# Nguyễn Trí Dũng Portfolio

Premium dark portfolio website built with React, Vite, TypeScript, Tailwind CSS, Framer Motion, React Router, and Lucide React icons.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite, usually `http://127.0.0.1:5173/`.

## Useful Commands

```bash
npm run build
npm run preview
```

## Customize

- Update portfolio content in `src/data/portfolio.ts`.
- Replace `public/nguyen-tri-dung-resume.txt` with your final resume PDF when ready.
- Replace GitHub and LinkedIn placeholder links in `src/data/portfolio.ts`.
